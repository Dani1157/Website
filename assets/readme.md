# User Experience (UX)

The goal of this website is to inspire users through authentic travel stories and visuals while keeping navigation simple and enjoyable.

## New User

As a first-time user, I want to:

* Easily understand the purpose of the website.
* Navigate between sections smoothly.
* Explore featured travel stories.
* View inspiring travel images.
* Contact the site owner through a simple form.

## Returning User

As a returning user, I want to:

* Discover new travel stories.
* Revisit favorite destinations.
* Explore updated gallery content.

# Deployment

The project was deployed using GitHub Pages.

### Deployment Steps:

1. Go to the repository on GitHub.
2. Navigate to **Settings**.
3. Select **Pages** from the side menu.
4. Under *Build and Deployment*, choose:

   * Source: Deploy from Branch
   * Branch: main
   * Folder: / (root)
5. Save and wait for deployment to complete.
